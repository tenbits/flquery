What's all this about?

CSS, HTML and jQuery for Flash Lite.

Why?(History)

It was one task: Application development for Samsung Bada Platform.
Possibilities:

  • C++ bada api development - no, thank you - too many code, difficult to build custom UI.
  • HTML5-based apps - rush development, but no, thank you - very low performance (css transitions and animations are almost unpossible).
  • Flash Lite apps - nativ performance feeling, and this project is aimed to bring many html/css/jquery goodies to flash/actionscript2

So, what is it?

  • DOM.
    element(body)
    element
    element
    element
    ...
  • Elements / Widgets.
    • div
    • span
    • input

  • Custom Elements/Widgets (with custom tags). Inherit from Div and create your element/widget (checkbox for example) and associate this element with a custom tag: bada.dom.NodesFactory.controls['checkbox'] = MyCheckboxClass
  • StyleSheets. Supported styles, see StyleSheet.styles
  • XmlParser.

    see more at Game.as
  • Selectors. Dom.body.find('#container').toggle(false);
  • Events. Dom.body.find('#container').bind('touchEnd', this.doSmth.bind(this)) Dom.body.find('#container').bind('customEvent', this.doSmth.bind(this)) Dom.body.find('#container').trigger('customEvent', argument1, argument2)
